🌐 ToolsLand BD

📖 About The Project

ToolsLand BD is a modern web platform where users can explore useful digital tools and services in one place. It focuses on simplicity, speed, and a smooth user experience, making it easy for users to find and use tools efficiently.

🚀 Technologies Used

• ⚛️ React

• 🎨 Tailwind CSS

• 📜 JavaScript

✨ Features

• 🔍 Browse Tools Easily — Clean UI to explore different tools quickly

• 🛒 Interactive Cart System — Add and manage selected tools/products

• 🔄 Dynamic Data Rendering — Tools are displayed dynamically using React
